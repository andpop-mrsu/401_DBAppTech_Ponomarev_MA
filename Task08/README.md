# Игра "Холодно-Горячо" - SPA версия

Веб-версия классической игры "Холодно-Горячо", реализованная как Single Page Application (SPA) с REST API сервером на PHP и базой данных SQLite.

## Требования

- PHP 7.4 или выше
- Веб-браузер с поддержкой JavaScript ES6+

## Установка и запуск

1. **Клонируйте или скопируйте проект**

2. **Запустите веб-сервер PHP:**
   ```bash
   # Перейдите в каталог Task08
   cd Task08
   
   # Запустите встроенный веб-сервер PHP на порту 3000
   php -S localhost:3000 -t public

   #Откройте приложение в браузере:
   http://localhost:3000/
